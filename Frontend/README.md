# Web3DRender Frontend

React.js frontend for the Web3DRender 3D visualization platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file (optional):
```
VITE_API_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

To build for production:
```bash
npm run build
```

## Features

- Modern, responsive UI matching Web3DRender design
- User authentication (Login/Signup)
- Project management
- 3D model upload and visualization
- Dashboard with statistics
- Multiple pages: Home, About, Pricing, Contact

## Tech Stack

- React 18
- React Router
- Vite
- Tailwind CSS
- Axios for API calls
- React Icons
